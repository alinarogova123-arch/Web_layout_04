# Киностудия «ГОСТФильм»

Сайт киностудии "ГОСТфильм".

## Как запустить

Для запуска сайта у вас уже должен быть установлен Python 3.

- Скачайте код
- Установите зависимости командой `pip install -r requirements.txt`
- Запустите сервер командой `livereload dist`

После этого переходите по ссылке [http://127.0.0.1:35729].

Пример, как выглядит сайт, доступен по следующей ссылке [https://alinarogova123-arch.github.io/Web_layout_04/dist/index.html]

Код написан в учебных целях — это урок в курсе по Python и веб-разработке на сайте [Devman](https://dvmn.org).
